Machine Learning Workflow
1. Data Collection
Data collection is the first and most important step in Machine Learning. It involves gathering data from different sources such as surveys, websites, databases, sensors, or files. The quality of the collected data directly affects the performance of the model. If the data is accurate and relevant, the model can make better predictions. The collected data should contain useful information related to the problem being solved. In this project, a student performance dataset is used. The dataset contains Study_Hours, Attendance, Assignments, and Marks.
2. Data Cleaning
Data cleaning is the process of preparing data before using it in a Machine Learning model. It involves checking for missing values, duplicate records, incorrect entries, and inconsistencies. Clean data helps improve the reliability and accuracy of the model. Without proper cleaning, the model may produce incorrect results. Data cleaning also makes analysis easier and more effective. In this project, the dataset does not contain missing values or duplicate records. Therefore, only basic verification of the data is required.
3. Feature Selection
Feature selection is the process of choosing the most relevant input variables from the dataset. These selected variables are used by the model to make predictions. Choosing useful features improves model performance and reduces unnecessary complexity. Good feature selection also helps in reducing training time. In this project, Study_Hours, Attendance, and Assignments are selected as features. These factors are expected to influence student performance. Therefore, they are used as inputs to predict marks.
4. Train-Test Split
Train-test split is used to divide the dataset into two parts: training data and testing data. The training data is used to teach the model, while the testing data is used to evaluate its performance. This helps in understanding how well the model works on unseen data. If all data is used for training, it becomes difficult to measure model accuracy properly. In this project, 80% of the data is used for training and 20% is used for testing. This is a common ratio used in Machine Learning projects.


5. Model Training
Model training is the stage where the Machine Learning algorithm learns patterns from the training data. During this process, the model studies the relationship between input features and the target variable. In this project, the Linear Regression algorithm is used. The model learns how Study_Hours, Attendance, and Assignments affect Marks. The trained model creates a mathematical relationship between these variables. This relationship is later used to make predictions. Proper training helps improve the overall performance of the model.
6. Testing
Testing is performed after the model has been trained. The testing dataset contains data that the model has not seen before. The model predicts values for this data and the predictions are compared with actual values. This helps determine how accurately the model performs. Testing is important because it shows whether the model can generalize to new data. A good model should perform well on both training and testing data. In this project, testing is carried out using the 20% test dataset.
7. Evaluation
Evaluation is the process of measuring the performance of the trained model. Various metrics are used to determine how accurate the predictions are. In this project, Mean Absolute Error (MAE) and R² Score are used for evaluation. MAE measures the average prediction error, while R² indicates how well the model explains the data. A lower MAE value means better prediction accuracy. A higher R² Score indicates a stronger model. Evaluation helps determine whether the model is suitable for practical use.
8. Prediction
Prediction is the final step in the Machine Learning workflow. After training and evaluating the model, it can be used to predict outcomes for new data. The model uses the learned relationship between features and the target variable. In this project, the model predicts student marks based on study hours, attendance, and assignments. Predictions help estimate future performance and support decision-making. Accurate predictions indicate that the model has learned useful patterns from the data. This is the main objective of Machine Learning.
